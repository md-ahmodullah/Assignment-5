Frequently Asked Questions

What is the Document Object Model (DOM)?

DOM stands for Document Object Model.

  • Document refers to an HTML or XML file; such as- "index.html", "blog.html"
  • Object refers to that JavaScript thinks of everything as an a tree of objects
  • Model refers to a workstation where can do manipulate content, structure, and styles dynamically.
    • In formal way, DOM is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, allowing developers to using languages like JavaScript.

    How do you manipulate an element's attributes and styles using the DOM?

    To manipulate an element's attributes and styles using the DOM, I will use methods like
    "setAttribute()" to set an attribute in an element;
    Ex: "buttonIdEl.setAttribute('onclick', showModal())
    "removeAttribute()" to remove an attribute from an element;
    Ex: "buttonIdEl.removeAttribute('onclick', showModal())"
    "element.style.property" directly modify properties
    Ex: "buttonIdEl.style.bg-lime-500" for inline CSS styles.

    What is event delegation in the context of the DOM, and why is it useful?

    An efficient way of event-handling is "Event delegation" reduces memory use, and supports dynamically added child elements.It attaches a single event listener to a parent element to handle events from its child elements.

    How do you select an element from the DOM?

    I can select a DOM element using methods like

  • getElementById()
  • getElementsByClassName()()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

    • to retrieve elements based on their ID, class, tag name or CSS selectors.
    Ex: "document.getElementById('history-button')"
    "document.getElementsByClass('donation-button')"
    Important Mentions:
    We can select a single element by using getElementById() & querySelector().
    We can select muliple element by using getElementByClassName(), getElementByTagName() & querySelectorAll().
